# 人格-学校-活躍経路 統合予測DB（PST-DB: Persona-School-Trajectory Integrated DB）

## プロジェクト概要

JPMS-DB（私立学校DB、551校・36,943件関係者の声・5層スキーマ）と era-talents-db（時代別活躍人材DB、12,958人物・47,284レコード・9時代×19能力次元）、およびフォーサイト関連DBを横断統合し、**入学前の子どもの人格モデル → 適合学校 → 活躍経路 → 歴史上の偉人パターンとの関連付け**という一連の予測モデルを構築する補助DB。

## 中核命題

子どもの人格特性は、(a) 適合的な学校教育環境を持ち、(b) その環境を通じて発達する能力次元が、(c) 時代の能力地形（過去〜現在〜未来）と相互作用することで、(d) 特定の活躍経路に至る蓋然性を持つ。歴史上の偉人は、その経路上に位置する具体的な事例である。ただし、時代性の差異により、偉人パターンを直接適用することはできず、現代・未来の能力地形に翻訳する必要がある。

## 4つの分析タスク

### タスク1：学校教育方針 × 時代が求める人物像（未来含む）の適合検証
- jpms_v2 schools_v2 (551校) × school_culture_score (5,280件)
- era-talents-db: era_discourses (L1) + era_retrospectives (L2) + future_demands (L4)
- 各学校の教育方針が、どの時代（明治〜2100）の能力像と最も適合するか定量化

### タスク2：学校×人格特性の心理学的モデル化
- jpms Layer 2: person_trait_dim 19件・person_archetype 10件（Big Five-J + Growth Mindset + SRQ）
- 心理学先行研究: Big Five・Holland Career Codes・MBTI批判・Adlerian・自己決定理論
- 各学校に最も適合する人格類型を心理学的に立体記述

### タスク3：予測モデル — 入学前の子どもの人格 → 適合学校 → 活躍経路
- 入力: 子どもの人格スクリーニング（Big Five-J+α）
- 出力: 適合学校 TOP-N + 各学校での活躍経路シミュレーション
- 数理モデル: jpms LCA k=8 + era-talents 偉人クラスタ + フォーサイト能力地形

### タスク4：歴史上の偉人パターン × 時代性配慮 × フォーサイト
- era-talents-db 12,958人を人格類型（Big Five+Holland）でクラスタリング
- 時代性差異（明治の偉人をそのまま現代に投影しない）の補正
- フォーサイト（2030/2050/2100）の能力地形と人格類型の適合予測

## 5層構造

PST-DB は次の5層で人格-学校-活躍経路を立体記述する。

| 層 | 内容 | データ源 |
|---|---|---|
| L1 子どもの人格 | Big Five-J + Holland + Growth Mindset | jpms person_trait_dim 19件 + 心理学拡張 |
| L2 適合学校 | 学校文化×教育方針×成果次元 | jpms schools_v2 551校 + culture_score |
| L3 活躍経路 | 学校→大学→職業→社会的影響 | jpms alumni_career 3,359件 + era-talents domain |
| L4 偉人パターン | 歴史上の同型人物 | era-talents-db 12,958人 |
| L5 時代適合 | 過去の偉人と現代/未来の翻訳 | era-talents L1/L2/L4 + フォーサイトDB |

## 倫理・限界の事前明示

- **「序列付け」禁止**: jpms の規律を継承、すべて「相性・適合」フレーム
- **決定論回避**: 「この子はこの学校に行くべき」ではなく「この子の傾向に合う環境はこういう特徴」
- **時代性の自覚**: 偉人を直接モデルにしない、「翻訳」が必要
- **データの限界**: jpms Layer 2 個人特性は実データ未充実、era-talents は記録バイアスあり
- **未来予測の不確実性**: シナリオ分岐を併記、単一予測を避ける
- **未成年データ**: anonymized_only、削除依頼SOP継承
