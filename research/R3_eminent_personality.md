# PST-DB 基盤先行研究レポート

## era-talents-db の12,958人を心理学的人格類型でクラスタリングする方法論

**執筆日**: 2026年5月6日  
**担当**: R3 偉人プロファイラー  
**対象DB**: era-talents-db (achievers: 12,958人 / achiever_capabilities: 31,430件)

---

## 1. 問題設定

PST-DB (Persona School Trajectory Database) は、人間の人格特性から職業選択、スキル獲得、キャリア軌跡へのパスウェイを描き出そうとする野心的なプロジェクトである。そのコア仮説は、「特定の人格プロフィールを持つ人間が、時代のアフォーダンスとの相互作用を通じて、一貫したキャリア軌跡を歩む」というものだ。

era-talents-db の12,958人の日本の活躍人材（明治1,412人〜令和4,382人）は、この仮説を検証する理想的なケーススタディを提供する。彼らは既に「成功」という選別を受けており、19次元の能力スコア（認知、価値観、社会性、行動力、創造性）を備えている。問題は、これらの能力次元スコアから、心理学的な人格特性（Big Five と Holland RIASEC）をいかに推定し、時代ごとの人格ポートフォリオを構築するかである。

本レポートは、その方法論的基盤を整理する。Simonton の historiometric personality assessment、テキストマイニングに基づく Big Five 推定、capacity-to-personality のマッピング理論、そして時代補正の枠組みを統合し、era-talents-db から PST-DB へのデータ変換パイプラインを設計する。

## 2. 理論的基盤：Historiometric Personality Assessment

### 2-1. Simonton による歴史心理学的評価

Dean Keith Simonton の historiometric method は、歴史的人物の人格を推定する古典的アプローチである。彼は、生没年、著作数、教えた学生数、受けた賞、文献言及頻度といった生涯データから、IQ や人格特性を逆算する手法を確立した。1986年から2014年にかけての一連の研究で、Simonton は以下を示唆している：偉人の知的能力は、伝記量（Wikipedia のページサイズ、文献引用数）から相関推定が可能であり、人格特性（特に Openness to Experience）は、多様な経験（mentors の数、居住地変更、教育的多様性）から推定可能である。さらに、リーダーシップ軌跡は、若年期の家族構成（喪失経験、多兄弟姉妹）と統計的相関を示すことも明らかにしている。

Simonton の方法論が era-talents-db に適用可能な理由は、我々がすでに以下を保持しているからである：人物ごとの family_class（貴族、武士、商人、農民、労働者）、education_path と mentors データ、domain（政治、ビジネス、科学、文化、スポーツ等）という専門領域タグである。これらのメタデータは、histometrics の基本変数群と対応している。

### 2-2. Big Five と歴史人物

Rubenzer, Faschingbauer & Ones (2000) は米国大統領39名を Big Five で評価し、leadership success との関連を示した。McCrae (1996) は文献から推定した Big Five スコアが、他者評価と有意な相関を示すことを確認している。この知見は、故人や歴史的人物の人格特性が、その業績記録から推定可能であることを示唆している。

era-talents-db との接点は多角的である。Openness to Experience（開放性）は、多ドメイン活動、mentors の多様性、新規領域への進出と相関する。Conscientiousness（誠実性）は、achievement_summary の「組織運営」「社会実装」への言及、継続性と対応する。Extraversion（外向性）は、政治、経営、スポーツ領域での活動、演説やメディア露出と相関する。Agreeableness（協調性）は、val_collective（集団協調性）スコア、社会運動や教育領域の活動と結びつく。Neuroticism（神経症傾向）は、負の側面が多いため、artifact として low neuroticism を推定するのが適切である。

era-talents-db の19次元能力スコアは、これら Big Five トレイトの「表現型」と見なすことができる。つまり、能力スコアは人格特性の行動的・認知的な現れ方を記述し、潜在的な人格特性への窓となるのである。

## 3. era-talents-db からの人格推定戦略

### 3-1. Capacity-to-Personality マッピング仮説

era-talents-db の19次元能力スコア（1-10の序数スケール）から Big Five への変換は、完全な推定ではなく、インディケータベースの仮定である。以下のマッピングを提案する。

創造性（cog_creativity）は高開放性（高O）の指標となる。新規性の追求と想像力が開放性の本質だからである。批判的思考（cog_critical）は高開放性と低協調性に寄与する。なぜなら批判的思考は規範への疑問と独立的判断を伴うため、必ずしも調和を重視しない。論理的思考（cog_logical）は高開放性と高誠実性の両者に寄与する。システマティックで厳密な思考は、開放性の思想的深さと誠実性の規則性の両面を反映している。

集団協調性（val_collective）は直接的に高協調性（高A）に対応する。寛容性（val_tolerance）は高協調性と高開放性の両者に寄与する。他者理解と価値観の多元性は、協調性の包容力と開放性の多元的価値観の双方を表現している。対人スキル（soc_interpersonal）は高外向性と高協調性に寄与する。影響力と感情読み取り能力は、外向性の社交性と協調性の共感性に基づいている。

起業家精神（age_entrepreneur）は高外向性、低協調性、高開放性の組み合わせを示す。起業家は野心的（高E）であり、個人的リスク選好（低A）があり、新規領域への志向（高O）を持つからである。レジリエンス（age_resilience）は低神経症（低N）を示す。ストレス耐性は不安感の低さに直結している。社会変革志向（age_social_change）は高外向性と低誠実性に傾く。既存秩序への異議や革新志向は、外向性の主張性と誠実性（既存ルール尊重）の相対的低さから生じるからである。異分野統合志向（cre_cross_domain）は高開放性の最たる例である。領域横断的思考は、固定的カテゴリの超越を意味し、これは開放性の本質である。

これらの関連性から、推定式（簡易版）を構築できる：

```
O_score = (cog_creativity + cog_critical + cog_ai_collab + cre_cross_domain) / 4
C_score = (cog_logical + cog_systems) / 2 + (age_meta_learning) / 1
E_score = (soc_interpersonal + age_entrepreneur + age_social_autonomy) / 3
A_score = (val_collective + val_tolerance + soc_interpersonal) / 3
N_score = 10 - (age_resilience + age_meta_learning) / 2
```

各スコアを5段階に変換し、最終的に "OCEAN" 5タプルを得る。これはスケーリングの恒等変換であり、統計的有意性を損なわない。

### 3-2. Holland RIASEC への domain マッピング

era-talents-db の domain タグから Holland RIASEC への対応を構築する必要がある。Holland の RIASEC モデルは、職業関心と人格特性の関連性に基づいており、6つのタイプ（Realistic, Investigative, Artistic, Social, Enterprising, Conventional）が特定の環境や職務に親和的である。

science_tech ドメインは Investigative タイプに対応する。科学者や技術者は分析的、論理的、システム志向的であり、抽象的問題解決を求める。business および entrepreneur ドメインは Enterprising タイプを示す。経営者や起業家は説得力、競争心、組織運営能力に優れている。culture_arts ドメインは Artistic タイプに対応する。文化人や芸術家は創造性、表現、個人的価値表現を重視する。politics および social_movement ドメインは Social / Enterprising の混合を示す。政治家や社会運動家は利他性と影響力の双方を持つ。craft ドメインは Realistic タイプに対応する。職人は実践的、技術的スキル、物質的操作を価値とする。education ドメインは Social タイプに対応する。教育者は教えることと発展支援に専念している。agriculture_local ドメインは Realistic タイプを示す。農業者や地域活動家は実践的、地域密着的である。

このマッピングにより、各人物に Holland code（例："IRA"、"SEI"）を割り当てることが可能となる。複数 domain に関わる人物については、primary と secondary の code を明示的に区別する。

## 4. テキストベース人格推定の補完層

era-talents-db の achievement_summary フィールドには、各人物の業績を簡潔に記述したテキスト（平均100-200字）が格納されている。LIWC（Linguistic Inquiry and Word Count）の方法論を応用することで、テキストベースの Big Five 推定が可能である。言語が人格を反映するという仮説は、言語学と心理学の交差領域で広く支持されている。

### 4-1. LIWC の知見から

メタ分析によれば、LIWC カテゴリと Big Five の相関は小〜中程度（ρ = .08 〜 .39）である。観察者評価（他者が述べたテキスト）では相関がより強く（ρ = .18〜.39）、自己報告より robust である。重要な言語マーカーとしては、Openness は長い単語、仮定法表現（「もし〜なら」）、分析的思考語に関連する。Conscientiousness は絶対的表現、因果語（「だから」「ために」）、人称代名詞の少なさに関連する。Extraversion は1人称複数代名詞（「我々」）と社交語に関連する。Agreeableness は肯定感情語の多さ、寛容表現に関連する。Neuroticism は1人称単数代名詞（「私」）と負感情語に関連する。

これらの関連性は、achievement_summary の日本語テキストに応用可能である。

### 4-2. 日本語テキストへの適用

era-talents-db の achievement_summary は日本語であるため、以下の日本語テキスト分析ツール/マーカーを活用する。形態素解析（MeCab / janome）で品詞分類を行い、感情極性辞書（豊田秀樹の感情語リスト等）で肯定/否定を分類する。複合語分析として、「革新」「実装」「運営」等の特定キーワードの出現頻度を計測する。

具体例として、"社会実装・組織運営" の言及は高 Conscientiousness を示す。"新規・革新・先駆" の言及は高 Openness を示す。"協働・連携・普及" の言及は高 Agreeableness を示す。これらのキーワードは、achievement_summary の記述者（歴史家や編集者）が該当人物を評価する際の言説パターンを反映しており、統計的に人格推定を補強できる。

## 5. 時代アフォーダンス補正フレームワーク

重要な前提がある。明治の起業家と令和の起業家の「人格」が同じ Big Five スコアを示すとしても、その社会的表現形は全く異なる。明治の entrepreneurship は、基本的に個人資本と冒険心からなるが、令和の entrepreneurship はデジタルプラットフォームと team leadership を求める。つまり、同じ人格特性が、時代というコンテクストに応じて異なる行動・成果として現れるのである。

### 5-1. 時代別「平均人格プロファイル」の計算

era-talents-db は6つの主時代を含む（明治、大正、昭和前期、昭和後期、平成、令和）。各時代のすべての活躍人材について、推定した Big Five スコアの平均値と分布を計算する。これにより、時代ごとの「典型的人格プロファイル」を得ることができる。

例えば、仮の計算結果として、明治時代の平均値は O=6.2, E=7.1, C=7.4, A=5.8, N=4.1 となり、令和時代は O=6.8, E=6.2, C=6.9, A=6.5, N=3.8 となるかもしれない。これは、明治が「外向性と誠実性の時代」（強いリーダーシップと規律）、令和が「開放性と協調性の時代」（多元性と連携）であることを示唆する。時代ごとの平均プロファイルは、その時代が求めた「理想的人格」の歴史的変化を記録する価値を持つ。

### 5-2. 「翻訳係数」の導入

個別の活躍人物の Big Five スコアを比較する際、時代差を補正するため、以下の翻訳係数を適用する：

Normalized_trait_t = (Raw_trait_t - Mean_trait_era) / StdDev_trait_era

これにより、各人物を「その時代の中での相対的位置づけ」として評価できる。例えば、明治の高 E (7.5) は、その時代の平均 E (7.1) からの逸脱度（+0.4）を示し、令和の高 E (7.2) の場合、平均 E (6.2) からの逸脱度（+1.0）と異なる。正規化することで、「明治では中程度の外向性」と「令和では顕著な外向性」が区別され、クロス時代比較が可能になる。

### 5-3. ドメイン内での時代差補正

さらに細かく、ドメイン × 時代のセルごとに平均プロファイルを計算することも可能である。例えば、「明治期の政治家」「昭和後期のビジネスパーソン」「令和のテック起業家」ごとに基準値を設定し、より文脈的な補正を行う。この階層化により、単なる時代差だけでなく、領域的な役割期待の変化をも制御できる。

## 6. クラスタリング設計：10タイプ × 6時代行列

### 6-1. 教師ラベルの確保

era-talents-db に存在する person_archetype（JPMS から応用された10人格類型、例：「リーダー型」「革新者型」「職人型」等）を準教師ラベルとして用いる。全12,958人がこのラベルを持つわけではないが、既存ラベルを持つ人物群に対して Big Five スコアの統計的プロフィールを構築することで、クラスタセンタを初期化できる。

この教師信号の強度は、JPMS データの品質に依存する。複数の独立した人物評価者がいれば、inter-rater reliability を検証し、一定の閾値以上の合意を持つラベルのみを教師として用いることが望ましい。

### 6-2. クラスタリング手法の選択

K-means クラスタリングは、目標クラスタ数を10に設定し、Big Five（5次元）+ Holland code（6次元の1-of-6 encoding）を特徴ベクトルとする。計算効率と解釈性に優れるが、球状クラスタ仮定があり、アウトライアーに敏感である。

ガウス混合モデル（GMM）は、各クラスタを多変量ガウス分布でモデル化し、確率的割り当て（soft assignment）により、周辺人物の重複を許容する。より柔軟だが、計算量が増加する。

トピックモデル（LDA）は、各人物を複数の「人格タイプの混合」として表現する。例えば、0.6 × リーダー型 + 0.3 × 革新者型 + 0.1 × 職人型 という混合度合いを捉えられ、era-talents-db の実データが多くの場合ハイブリッド的であることに適合する。

推奨する戦略は、K-means + GMM の2段階である。まず K-means で粗い初期化を行い、その後 GMM で精密化する。これにより、高速な初期配置と柔軟な最適化を両立できる。

### 6-3. 3次元マトリックス構造

最終的に、以下の構造を得る。時間軸（era: 6次元）× 人格軸（person_archetype: 10次元）× ドメイン軸（14ドメイン）の3次元行列として、各セルが以下の情報を保有する：メンバー数、Big Five プロファイル（平均値 ± 標準偏差）、Holland code 分布、代表人物（各類型3-5名）、典型的達成パターン、キャリア軌跡プロトタイプ。

例として、[Meiji × リーダー型 × Politics] のセルは、142名を含み、O=6.4, E=7.8, C=7.6, A=5.2, N=3.9 を示し、Holland code は SE（Social-Enterprising）を主とするかもしれない。一方、[Reiwa × 革新者型 × Science/Tech] のセルは、87名を含み、O=7.9, E=6.1, C=6.8, A=6.2, N=4.2 を示し、Holland code は IA（Investigative-Artistic）を主とするかもしれない。このマトリックスから、時代別・タイプ別・領域別の人格プロファイルの進化を読み取ることができる。

## 7. 限界と倫理的考慮

### 7-1. 死者の人格推定の妥当性

Historiometric assessment の最大の課題は、故人の実在した人格を推定できるかということである。我々が保有するデータは、二次文献（伝記、百科事典）の記述に基づいており、故人が意図的に構築した公的イメージが反映されている可能性があり、時代の記録者によるバイアスがある。

対策としては、複数の伝記ソースからの三角測量を行うことだが、当該レポートでは実装していない。また、推定スコアの信頼区間を明示し、「確定値」ではなく「仮説」として扱う必要がある。最も重要なのは、敏感な解釈（「この人は神経症傾向が高い」等）を避け、「社会変革志向が強い」等の行動記述に留めることである。人格推定を人物評価に転換しないという倫理的警戒が必須である。

### 7-2. 西洋中心の Big Five バイアス

Big Five（OCEAN）は、1980年代の米国の心理学研究から抽出されたモデルである。日本を含む非WEIRD（西洋・教育水準高・先進国・民主的）文化における妥当性は、相対的に低い。研究によれば、Big Five が有効な測定を与えるのは、西洋サンプルで80%以上だが、非西洋サンプルではわずか45%程度である。

特に問題となる領域がある。Agreeableness は、日本的文脈では「調和」「空気を読む」がより複雑で、西洋的「個人的親切」とは異なる。Conscientiousness は、過度な自己規律より「義務感」「役割遂行」が強調される文化では、測定が不適切になる。Openness は、日本では「伝統尊重」と「革新」のバランスが文化的価値であり、一次元では捉えきれない。

対策としては、Big Five を「参照フレーム」として用いつつ、日本的な概念体系（儒教的徳性、禅的素質観、心（こころ）の伝統的分類）との対照を常に意識すること。将来的には、era-talents-db の日本的文脈での人格類型体系（例：福沢諭吉の「独立」vs 西郷隆盛の「義」といった歴史的人格アーキタイプ）の独立開発も検討する価値がある。

### 7-3. ジェンダー表現の歪み

era-talents-db の achievers の約90%が男性である（women_pioneers domain を除く）。これは、歴史的な機会喪失の反映として正当であるが、同時にデータ選別アルゴリズムの潜在的バイアスの可能性も排除できない。結果として、10個の person_archetype プロファイルが「男性中心」になる可能性が高い。

対策としては、女性活躍人材のサブクラスタリングを別途実施し、gender-specific career trajectory patterns を明示的に可視化する必要がある。また、データ収集段階で女性の過小代表がどの領域で著しいか（例：政治で極度に低い、教育では相対的に高い）を明記し、解釈時の文脈を提供する。

### 7-4. 前向き推定 vs 後向き推定の混同

era-talents-db は「すでに成功した人物」を対象としている。つまり、「O=7, E=6 という人格は科学者になりやすい」という知見は、生存バイアス（survivor bias）に制約される。O=7, E=6 の多くは確かに科学者になったが、O=7, E=6 で科学者にならなかった人は、このデータセットに含まれていない。

対策としては、PST-DB の適用において、「この人格なら確実に科学者になる」という決定論的言述を避けることである。「この人格傾向を持つ顕著な成功者が科学領域に多く見出される」という条件付き表現に留めることが重要である。また、可能であれば、統計的に非成功者のデータ（試行錯誤、キャリア転向、離職等）を追加し、両側の推定を行う方が誠実である。

## 8. PST-DB 実装への含意と次ステップ

### 含意1: Big Five スコア生成パイプライン

era-talents-db の全12,958人に対し、19次元能力スコア → Big Five スコアの自動変換スクリプトを構築する。実装は Python ベースで、実装難度は中程度である。時代別・ドメイン別の正規化ロジックを組み込み、各人物の raw score から normalized score への変換を行う。このパイプラインは idempotent であるべき（何度実行しても同じ結果）で、version control に入れ、変換ロジックの透明性を保つ。

### 含意2: Holland Code の割り当て

achievers.domain フィールドから Holland RIASEC への確定的マッピングを行う。14ドメイン × 6 RIASEC type の対応テーブルを構築し、各人物に「primary」「secondary」code を付与する。複数領域で活動した人物については、主要活動領域を primary とし、副次領域を secondary とする。これにより、各人物のキャリアパス（初期 domain → 最終 domain）において Holland code がどのように変化するかを追跡できる。

### 含意3: テキスト分析補完層

achievement_summary および notable_works フィールドに対し、日本語 LIWC 相当の分析を実施する。感情語抽出、キーワード出現（「革新」「実装」「協働」等）を計測し、Big Five スコアの robust 化とクロスバリデーションを行う。テキスト分析が推定スコアと矛盾する場合（例：achievement_summary で「革新」語が多いが推定 O が低い場合）、推定アルゴリズムの改善点を指摘できる。

### 含意4: 準教師クラスタリング + 可視化

既存 person_archetype ラベルを用いて、K-means または GMM でクラスタセンタを初期化する。その後、ラベルなし人物についても確率的割り当てを行う。結果を3D プロット（Big Five 空間）や network graph（人物間相似度 edge）で可視化し、クラスタ構造の妥当性を視覚的に検証する。interactive visualization（例：Plotly, D3.js）を用いれば、ユーザーがクラスタを動的に探索でき、PST-DB の理解が深まる。

### 含意5: キャリア軌跡パターン抽出

10タイプ × 6時代のセルごとに、典型的なキャリア軌跡を narrative で記述する。「令和の高O-高E型がどのように知識と社会実装を統合しているか」といった時系列的解釈を加え、単なる統計集計から意味的な理解へと昇華させる。例えば、「明治の低O-高E型は、既存制度の効率化に専念し、大正・昭和へ向けて新規領域開拓が限定的だった」といった歴史的観察が可能になる。

## 結論

era-talents-db の12,958人の日本の活躍人材から心理学的人格類型を推定することは、従来の historiometric 手法（Simonton）、テキスト分析（LIWC）、および能力次元マッピングを統合すれば、統計的に根拠のある形で実行可能である。この方法論は、歴史データの豊富さ（meta data、achievement_summary、capability scores）を活用し、心理学的個性化への道を切り開く。

しかし同時に、「死者の人格推定」「西洋心理学モデルの日本的適用」「生存バイアス」といった根本的な限界を完全に克服することはできない。重要なのは、これらの限界を透明化し、PST-DB の利用者に明示することである。一見客観的に見える数値化も、実は多くの仮定と選択を内含している。その全体的な前提を理解してこそ、初めて PST-DB は有用なツールとなる。

PST-DB は、「この人格なら確定的にこのキャリアに至る」という決定論ではなく、「この時代・ドメイン・人格傾向の組み合わせで、どのような活躍経路が出現する傾向があるのか」という、確率的・歴史的・文化的に適応的な理解枠組みとして機能すべきである。それは、人間の自由意志と社会的制約の相互作用を見つめる、謙虚で誠実なスタンスを要求する。

次段階は、提案した推定パイプラインの prototype 実装、validation（既知 person_archetype との一致度）、補正係数の fine-tuning である。並行して、日本的人格類型体系との比較研究、女性活躍人材のサブクラスタリング、歴史的 narrative の収集という三層の拡張も計画されるべきである。
